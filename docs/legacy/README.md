# Legacy Materials

Earlier experimental materials, raw manuscripts, legacy runtime configs, and abandoned
side projects have been moved out of the main branch to keep this repository focused
on AI Company OS as a product and evidence layer.

These materials are preserved in local backups. Key historical design documents that
still hold conceptual value have been archived under `docs/constitution/legacy-archives/`.

## Where to find what

| If you're looking for... | It's now at... |
|:-------------------------|:---------------|
| Old agent architecture designs (AGENTS, SOUL, TOOLS, HEARTBEAT, ROUTING) | `docs/constitution/legacy-archives/` |
| novel-v1 raw manuscripts (full chapter drafts) | Local backup only — summary at `examples/novel-v1/` |
| Hermes skills (50+ skills) | Local Hermes installation (`~/.hermes/skills/`) |
| Pre-v0.2 archive (experiments, old projects, book drafts) | Local backup only |
| Old build logs (pre-v0.2 era) | `docs/build-logs/legacy/` |
| v0.1.1 release reports | `docs/legacy/reports/` |

## Why

This repository's primary purpose is now a product repository + public evidence layer + operating kit entry point for AI Company OS. Historical materials that don't serve these three purposes have been moved out to keep the repository focused, searchable, and approachable for new visitors.
