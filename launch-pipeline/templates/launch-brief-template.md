# Launch Brief 模板

> 每个产品上线前，填写此模板 → 交予 Launch Pipeline

---

## 1. 产品基础信息

| 字段 | 值 | 说明 |
|:----|:---|:-----|
| `launch_run_id` | LR-XXX | 自动生成，唯一编号 |
| `product_name` | | 产品名称 |
| `product_type` | | 服务 / 方法论 / SaaS / 课程 / 电子书 / 工具 / 娱乐 |
| `product_line` | | 所属产品线（ai-company-os / ai-seller-finance / amazon-business / digital-products / launch-sites / entertainment-products） |
| `version` | | 版本号 |
| `launch_date` | | 目标上线日期 |

## 2. 目标用户

| 字段 | 值 |
|:----|:---|
| `target_user` | 一句话描述目标用户 |
| `pain_point` | 核心痛点（1-3 条） |
| `user_scale` | 估算用户规模 |

## 3. 产品价值

| 字段 | 值 |
|:----|:---|
| `offer` | 核心价值主张（一句话） |
| `differentiator` | 差异化（和竞品/替代方案的区别） |
| `proof_points` | 数据 / 案例 / 证据点 |

## 4. 定价与转化

| 字段 | 值 |
|:----|:---|
| `price_tiers` | 定价层级（免费 / ¥xx / ¥xxx / 年付） |
| `sample_asset` | 样品 / 体验版路径 |
| `cta` | 主要 Call to Action |
| `conversion_goal` | 预期的转化指标 |

## 5. 销售与分发（预留字段）

| 字段 | 值 | 说明 |
|:----|:---|:-----|
| `distribution_channel` | | 主分发渠道（自有站 / 小红书 / 微信群 / Gumroad / 知乎 / 公众号） |
| `cold_start_copy` | | 冷启动文案方向（发布时用） |
| `target_platform` | | 目标发布平台 |
| `success_metric` | | 成功指标（体验官数 / 付费用户 / 访问量 / 转化率） |
| `next_promotion_action` | | 发布后的下一步推广动作 |

## 6. 部署

| 字段 | 值 |
|:----|:---|
| `deployment_target` | Cloudflare Pages / VPS / GitHub Pages / 其他 |
| `deployment_domain` | 目标域名 |
| `environment_vars` | 所需环境变量清单 |

## 7. 额外说明

```
自由文本：说明本次 launch 的特殊需求、已知风险、期望的验收标准
```
