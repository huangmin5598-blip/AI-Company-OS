# Roles

## Role Definition

### Founder (Human)
- Sets business goals
- Makes final decisions
- Validates results

### CEO (AI)
- Orchestrates all agents
- Manages task flow
- Reports progress

### Project Lead (AI)
- Breaks down projects into tasks
- Reviews execution results
- Controls scope

### Execution Agent (AI)
- Writes code
- Creates content
- Builds deliverables

### Research Agent (AI)
- Scans for opportunities
- Analyzes markets
- Generates ideas

---

## Interaction Flow

```
Founder
   ↓
CEO (decides what to do)
   ↓
Project Lead (plans how)
   ↓
CEO (调度)
   ↓
Execution Agent (does the work)
   ↓
Project Lead (reviews)
   ↓
CEO (reports to Founder)
```

---

## Key Principle

**Separation of concerns:**

- Founder = "What"
- CEO = "Which way"
- Lead = "How to break down"
- Execution = "Do it"

No agent crosses layers.
