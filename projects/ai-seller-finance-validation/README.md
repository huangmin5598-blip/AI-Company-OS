## 创始决策日志

### 2026-05-28：OP-006 正式从 Opportunity 转为 Validation Project

**决策内容：**
- OP-006 **不是新产品**，而是 AI经营系统 v0.1 的商业化包装/销售入口
- OP-005（Amazon 利润体检报告）是首个付费验证切口
- 当前阶段只做**验证准备** —— 文档、能力映射、报告模板、LLM 方案，**不开发完整 SaaS**

**不做清单：**
- ❌ 不开发 AI经营系统 v0.2
- ❌ 不写 Settlement 解析引擎
- ❌ 不接 SP-API
- ❌ 不接完整真实 LLM
- ❌ 不改 5 个 Skill
- ❌ 不做多平台
- ❌ 不做新前端页面
- ❌ 不改 AI Company OS runtime
- ❌ 不做 v0.10 生产化
- ❌ 不做 CEO Agent 飞书上线

**完成标志：** 冷启动有付费信号 → 再进入 abnormal_diagnosis 真实 LLM 接入 / Settlement 解析最小版本 / 上传版报告生成

### 2026-05-29：GPT 建议进入 Validation Project 准备阶段

采纳要点：
- 执行顺序：目录 → 能力映射 → 报告模板 → LLM 方案（按优先级）
- Provider-agnostic LLM Gateway 设计（不锁死某一家）
- AI-Knowledge-OS 同步长期记录，但主工作目录在 ai-company-os repo
