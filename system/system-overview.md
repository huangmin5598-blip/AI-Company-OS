# System Overview

## What is AI Company Model

AI Company Model is an ongoing experiment exploring whether a solo founder can organize AI agents into a company-like operating structure.

It is not a product, platform, or service.
It is an operating model under active construction and validation through real project execution.

## System Structure

Founder  
↓  
CEO Agent  
↓  
Project Leads  
↓  
Shared Capability Agents

## Why This Repository Exists

This GitHub repository serves as the Evidence Layer of AI Company Model.

Its purpose is to document real system runs and preserve verifiable records of how the operating model works in practice.

It records:
- system structure
- experiment records
- project records
- deliverable evidence

It is not:
- a marketing platform
- a content platform
- a methodology platform

It is:
- a factual record of what actually happened
- evidence of system capabilities and limitations
- a basis for iteration and improvement
