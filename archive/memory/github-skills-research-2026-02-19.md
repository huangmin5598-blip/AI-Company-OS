### 1. Agent Skills 是什么

- 2025年12月由 Anthropic 发布为开放标准
- OpenAI 也采用了相同格式（Codex CLI）

## 四、AI 赚钱相关

### 热门方向
1. **AI 咨询服务** - 帮助企业自动化营销、优化流程
2. **AI 提示工程** - 为企业优化 AI 工具
3. **AI 驱动的内容创作** - 社交媒体、AI 生成图片
4. **自动化服务** - 邮件、客服、数据处理自动化

### GitHub 有趣项目
- **Avanzai** - AI agent 自动分析新闻、实时数据、投资组合风险
- **500-AI-Agents-Projects** - 500个 AI agent 应用案例（医疗、金融、教育、零售）
- **Multi-Agent RFP Reviewer** - AI 审核 RFP 文档

## 五、值得关注的技能类型

1. **编码类** - 代码审查、重构、测试生成
2. **研究类** - 网络搜索、竞品分析、数据抓取
3. **自动化类** - 浏览器自动化、邮件处理、日程管理
4. **金融类** - 股票分析、加密货币、自动化交易
5. **内容类** - 社交媒体运营、内容生成

## 六、趋势总结

- **MCP (Model Context Protocol)** 成为标准 - AI 工具互联
- **Agentic Workflow** - AI 自主规划、执行、反馈的工作流
- **低代码/无代码** 平台崛起 - n8n, Activepieces
- **技能市场** 兴起 - ClawHub, SkillsMP, Agent Skills Marketplace
