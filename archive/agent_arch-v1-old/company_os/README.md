# OpenClaw Company OS Pack v3

这是基于《OpenClaw 多项目 AI 机器人公司总架构蓝图 V1.1》生成的增强版配置草案包。

## 包含内容
- configs/: 主配置、agents、bindings、models、tools
- agents/: Agent 目录、SOUL.md、config.yaml
- registries/: 项目/渠道/能力注册表
- docs/: 部署与接线手册、实施说明

## 核心模型策略
- CEO / 项目主管: MiniMax 2.5
- 研发中心: 阿里云 Coding Plan
- 短剧平行稿: 本地 Qwen / DeepSeek / 千问
- 小说衍生: Qwen Planner + DeepSeek Writer + Qwen Editor
- 重大决策: 人工与 ChatGPT 讨论
