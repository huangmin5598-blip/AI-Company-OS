# Novel-v1 Copyright Policy (Global)

## Core Principle
Only imitate "genre patterns", never imitate "specific works".

## Allowed
- Common tropes (romance, revenge, identity swap, marriage contract)
- Generic emotional structures
- Industry-standard plot frameworks

## Forbidden
- Referencing specific novels, authors, or IP
- Rewriting identifiable plots from known works
- Using unique world-building from existing IP
- Using recognizable character systems or organizations

## Red Flag Rule
If a story can be described as:
"This is basically <specific novel> with slight changes"

→ MUST be rejected.

## Risk Levels
- **S-Level**: Strictly Forbidden (typical IP structure)
- **A-Level**: Avoid in Phase 1 (high template overlap)
- **B-Level**: Controllable (common tropes, need differentiation)
- **LOW**: Generic genre fiction (SAFE)

## Enforcement
- Enforced at: lead-novel, story-editor, writer, review-editor
- Final authority: review-editor

## Four-Layer Protection
```
Title Guard
↓
Keyword Risk Check
↓
Agent Self Check
↓
Review Editor Final Gate
```
