## 使用说明

### 生成工具推荐
1. **DALL-E 3** (ChatGPT Plus)
2. **Midjourney** 
3. **Stable Diffusion** (本地)
4. **Canva AI**

### 生成后处理
1. 用 Photoshop/Canva 添加透明圆形遮罩
2. 确保宠物照片可叠加到遮罩位置
3. 导出 1024x1024 PNG

### 叠加方式
```css
.pet-meme {
  background-image: url('template.png');
}
.pet-photo {
  width: 300px;
  height: 300px;
  border-radius: 50%;
  position: absolute;
  top: 50%;
  left: 50%;
  transform: translate(-50%, -50%);
}
```
